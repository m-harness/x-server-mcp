# 要件定義

## プロジェクト概要

エックスサーバーの REST API を MCP（Model Context Protocol）サーバーとして公開するプロジェクト。
AIエージェント（Claude等）がエックスサーバーの管理操作を実行できるようにする。

## 目的

- エックスサーバーの管理作業をAIエージェント経由で自動化
- サーバーパネルにログインせずに、自然言語指示でサーバー管理が可能に
- 反復的な管理タスク（メールアカウント作成、ドメイン追加等）の効率化

## Xserver API 概要

### 基本情報

| 項目 | 値 |
|------|-----|
| ベースURL | `https://api.xserver.ne.jp` |
| ベースパス | `/v1/server/{servername}` |
| プロトコル | HTTPS |
| レスポンス形式 | JSON |
| 成功ステータス | 200（全メソッド共通） |

### 認証

- `Authorization: Bearer xs_xxxxxxxxxxxx...` ヘッダーで認証
- APIキーはXServerアカウントの「APIキー管理」から発行
- キー名・有効期限・対象サーバーアカウント・権限を設定可能

#### 権限（スコープ）

| 権限タイプ | 利用可能API |
|-----------|------------|
| すべての操作（full） | 読み取り + 書き込みのすべてのAPI |
| 読み取り専用（read） | 読み取りのAPIのみ |
| カスタム（custom） | 個別に選択した権限のAPI |

### レート制限

レスポンスヘッダーで制限情報が返される。

| ヘッダー | 説明 |
|---------|------|
| `X-RateLimit-Limit` | 分あたりの上限 |
| `X-RateLimit-Remaining` | 残りリクエスト数 |
| `X-RateLimit-Reset` | リセット時刻（Unix timestamp） |
| `X-RateLimit-Concurrent-Limit` | 同時接続数の上限 |
| `X-RateLimit-Concurrent-Remaining` | 残り同時接続数 |

制限超過時は HTTP 429 + `Retry-After` ヘッダー。

| プラン | req/分 | req/日 | 同時接続数 |
|--------|--------|--------|-----------|
| スタンダード | 60 | 10,000 | 5 |
| プレミアム | 120 | 30,000 | 10 |
| ビジネス | 300 | 100,000 | 20 |

### エラーレスポンス

```json
{
  "error": {
    "code": "VALIDATION_ERROR",
    "message": "入力値が正しくありません",
    "errors": ["エラーメッセージ1"]
  }
}
```

| エラーコード | HTTP | 説明 |
|-------------|------|------|
| （コードなし） | 400 | リクエストが不正 |
| UNAUTHORIZED | 401 | APIキーが未指定・無効・期限切れ |
| FORBIDDEN | 403 | 権限不足またはIP制限 |
| NOT_FOUND | 404 | リソースが存在しない |
| OPERATION_ERROR | 409 | サーバー側制約により操作不可 |
| VALIDATION_ERROR | 422 | バリデーションエラー |
| RATE_LIMIT_EXCEEDED | 429 | レート制限超過 |
| INTERNAL_ERROR | 500 | API内部エラー |
| BACKEND_ERROR | 502 | バックエンド通信エラー |

### 共通仕様

- **サーバー名（servername）**: 初期ドメインを指定（例: `xs123456.xsrv.jp`）
- **ドメイン所有権確認**: 一部API（ドメイン追加・メールアカウント作成）で `_xserver-verify.{domain}` TXTレコード検証が必要
- **日本語ドメイン**: ドメイン追加以外はPunycodeで指定

## 共通技術ルール

- ES Modules（import/export）を使用。CommonJSは禁止
- stdoutはJSON-RPC通信用のため `console.log()` 禁止。デバッグは `console.error()` + ファイルログ
- APIキー等の機密情報はコードに含めない。環境変数 `XSERVER_API_KEY` から取得
- .envファイルの内容は読み取り・出力・転写しない
